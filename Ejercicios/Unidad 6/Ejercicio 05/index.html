Crea una página web que tenga un TextArea, una lista desplegable con colores y un botón "Crear párrafo". Cuando el usuario haga clic sobre el botón se creará un nuevo párrafo que contendrá el texto del TextArea y el color del fondo será el seleccionado en la lista desplegable. En caso de que el TextArea esté vacío, mostrará un mensaje de error y no creará ningún párrafo.
